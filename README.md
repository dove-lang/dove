### Usage ğŸƒ <a name="usage"></a>
To start using Dove, please follow these steps:

#### Online Playground <a name="usage-playground"></a>
Visit â†’ https://dove-lang.com/playground.

#### Linux <a name="usage-linux"></a>
1. Install the Dove interpreter binary â†’ [v0.1.2](https://github.com/dove-lang/dove/releases/download/v0.1.2/dove)
2. Navigate to the file directory.
``` 
cd [DIRECTORY]
```
3. Run the REPL, or run a dove file directly.
```
# Run REPL
dove

# Run file
dove example.dove
```

#### Windows <a name="usage-windows"></a>
1. Install the Dove interpreter binary â†’ [v0.1.2](https://github.com/dove-lang/dove/releases/download/v0.1.2/dove.exe)

2. Navigate to the file directory.
``` 
cd [DIRECTORY]
```

3. Run the REPL, or run a dove file directly.
```
# Run REPL
dove.exe

# Run file
dove.exe example.dove
```
